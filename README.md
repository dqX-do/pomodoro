# 🍅 番茄钟 (Pomodoro Timer)

一个漂亮的番茄钟 Web 应用，纯前端，双击即用。

## 功能

- ⏱️ 25 分钟专注 + 5 分钟短休 + 15 分钟长休
- 🔄 SVG 环形进度动画
- 🔊 完成时声音提醒 + 桌面通知 + 中文语音播报
- ⌨️ 快捷键：`Space` 开始/暂停，`R` 重置
- 📋 任务清单（保存到浏览器本地）
- ⏰ 工作时防止屏幕休眠 (Wake Lock API)
- 📊 番茄完成数自动统计

## 使用方法

直接在浏览器打开 `index.html` 即可，无需任何服务器。

## 技术栈

纯 HTML + CSS + JavaScript，零依赖。
